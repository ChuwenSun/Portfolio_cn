Verse Monk音乐社区

Verse Monk是一个为音乐爱好者提供音乐社区的Web应用。它以广大歌词库为基础，给音乐发烧友们提供给歌词细节提供注释和评论的功能来达到社交的目的。
在热榜和全部歌曲界面，用户可以和数据库内的歌曲信息进行互动。
开发了过滤分类功能，数据库里的歌曲可以根据用户需求以不同的音乐类型，语言，和年代进行分类。图例中显示为所有90年代的法语说唱歌曲。
在查看一首歌的歌词时，用户可以拖选一段歌词来添加注释，和其他用户分享自己独到的见解。
被添加过注释的歌词部分会带有灰色背景，点击他们可以查看其他用户添加的注释。
后端服务器由Node.js开发，由一套REST API提供对存在MongoDB里的数据库提供CRUD操作。
❮ ❯

Verse Monk 是一个旨在为全球音乐爱好者提供一个关于歌词细节和音乐多样性的社区的网络应用。用户可以编辑歌词，添加注释和解释歌词中的小细节，并与社区互动。这款应用受 Genius.com 的启发，目前正在努力重现其主要功能。项目进展仍然在每天推进。

相关技术工具: Angular 17, MongoDB, Node.js, Tailwind CSS, Express, Postman, Sequelize, REST API

开发中的功能: 搜索(Elasticsearch), 注释功能的OOP开发, 前端润色

游戏: 背包客的旅行日记

《背包客的旅行日记》是一款基于Low-poly美术风格，采用程序化内容生成技术的无边界野外探险游戏。
游戏的开始菜单将以首次生成的游戏大陆俯视图为背景。
截图为随机生成的自然景观和植被。游戏氛围轻松解压。
12 种独特的动物，它们具有与玩家和环境互动的 AI 行为。
靠近浣熊会把小家伙吵醒！
拍摄你发现的动物照片，并将它们放入你的旅行日记中。
用高分照片填满你的日记吧！
❮ ❯

背包客的旅行日记是一款无限开放世界的探险游戏，旨在为玩家提供一种轻松解压的游戏体验。游戏的主要目标是拍摄尽可能精彩的动物和植物照片。游戏采用美观的Low-poly, Flat-shaded的艺术风格。地形生成和植物生成系统基于 Perlin Map和Seeding系统，可以调整山脉、水域、植物或岩石出现的频率。存储和加载系统是基于 XML 文件的系统，用于保存或加载游戏信息的 XML 文件。

相关技术工具: C#, Unity

UberMate打车热点推荐系统

Ubermate是一个大数据pipeline被构建用于自动处理出租车/优步行程的原始数据，以输出帮助司机提升接单率和收入的热点推荐。
数据可视化以热力图的形式生成。此示例使用了纽约州的数据。红色区域表示订单量高的地方。.
后台日志显示后端程序交替自动运行，并依次进行pipeline中清理、预处理和分析阶段。
后台日志显示后端程序交替自动运行，并依次进行pipeline中清理、预处理和分析阶段。
❮ ❯

在这个项目中，我们设计了一个在 Linux 上运行的全自动大数据pipeline，可以接收流式的原始行程数据并为司机生成热点推荐，以最大化利润和接单效率。该管道将接收原始的 Parquet 文件并将其导入 HDFS 存储，然后进行清理和预处理。之后，完全准备好的数据将被导入 Hive 数据仓库。K-Mean和 PageRank 算法将接收信号分析新数据，并将结果输出到 Tableau 进行可视化。

相关技术工具: Python, Apache Spark, Apache Hive, Apache NIFI, Hadoop, K-mean, Pagerank, json, Tableau
