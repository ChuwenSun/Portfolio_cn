Verse Monk

Verse Monk is a web application of a music community based on a lyrics database, offers its users the abilities to adds annotation to lyrics and interact with the community.
User can interact with the song-info database in the Hot songs page or All songs page.
A filter is implemented to filter the query of songs by music genre, years and languages. For example, the screenshot above shows all french rap songs in the 90s.
When viewing the lyrics of a song, user can add annotation to sentences to let other people know detail or back story of the lyrics.
Lyrics with annotations will have a grey background, click them to view the annotation.
A backend server is maintained on Render.com with a REST API that allows connection to a MongoDB database.
❮ ❯

Verse Monk is a web application project that is aimming to provide a global music community for people that are interested in details of lyricism and diversity in music. Users may edit song lyrics, adding annotation and explanation to small parts of lyrics in a song and interact with the community about it. This app is inspired by Genius.com and currently trying to recreate the main functionalities of it. Progress is still been made on a daily basis.

Technologies: Angular 17, MongoDB, Node.js, Tailwind CSS, Express, Postman, Sequelize, REST API

Game: The Backpacker's Journal

The Backpacker's journal is a low-poly style, procedurally generated and infinite(Boundaryless) world adventure game.
Start menu with the bird-eye view of the firstly-generated continent

Capture photo of the animals you found and put them into your journal.
Complete your journal with high-scoring photos.
❮ ❯

The Backpacker's Journal, is an infinite, open-world safari game intended to just be a chill, low-stakes escape from the hardships of life. The main goal of the game is to take the best pictures possible of interesting animals and flora in the game. The game is intended to have an aesthetically pleasing low-poly, flat-shaded art style. The world-generation and flora-generation system are based on a Perlin Map and seeds system where we can change the frenquency of mountains, water, floras or rocks. The Save & Load system is a XML-based system that will save or load XML files for important information.

Technologies: C#, Unity

UberMate Hotspot recommendation System

This big data pipeline was built to automatically process raw data of taxi/uber trips to output hotspot recommandation for drivers
Data visualization was generated as heatmap. New York State data was used for this example. Red spots means high volumes of orders.
Logs showing that backend programs runs automatically and taking turns between cleaning, preprocessing, analysing phases.
Logs showing that backend programs runs automatically and taking turns between cleaning, preprocessing, analysing phases.
❮ ❯

In this project, a big data fully-automated pipeline is designed on Linux that can take streaming raw trip data and generate hotspot for drivers to maximize profit. The pipeline will take raw parquet files and ingest them into HDFS storage, and then be cleaned and preprocessed. And then the fully-prepared data will be ingested into the Hive data warehouse. K-mean and Pagerank algorithms will receive signals to analyze new data and output to Tableau for visualization.

Technologies: Python, Apache Spark, Apache Hive, Apache NIFI, Hadoop, K-mean, Pagerank, json, Tableau
